接口说明
语音合成流式接口将文字信息转化为声音信息，同时提供众多特色发音人（音库）供选择。该能力通过 WebSocket API 提供，具备流式传输能力，适用于需流式数据传输的 AI 服务场景。相较于 SDK，API 轻量、跨语言；相较于 HTTP API，WebSocket 原生支持跨域。
原 WebAPI 普通版本接口（http [s]://api.xfyun.cn/v1/service/v1/tts）不再对外开放，已使用用户仍可使用，建议迁移至新版流式接口。
小语种及少数民族方言：新增韩语、日语、维吾尔语、藏语、法语、俄语、西班牙语、印地语、德语、越南语、巴西葡萄牙语、意大利语、葡萄牙语、泰语、乌尔都语；使用小语种上传文本推荐 UTF-8 编码，需先在控制台开启对应小语种发音人，否则报错 11200。
接口 Demo
示例 demo 可在官网下载，目前提供 Java、Python3、JS、NodeJS 等语言 demo，其他语言参照文档开发。

请求行	GET /v2/tts HTTP/1.1
接口鉴权	签名机制

接口调用流程
通过接口密钥基于 hmac-sha256 计算签名，发送 WebSocket 握手请求。
握手成功后，客户端通过连接上传数据并接收音频流。
数据上传完毕，上传数据结束标识。
收到服务器返回结束标识（status=2）后断开连接。
WebSocket 注意事项
服务端支持 websocket-version 13。
服务端返回帧类型均为 TextMessage（opcode=1）。
分帧问题需升级框架或更换方案。
会话结束尽量使用错误码 1000 关闭连接。
白名单
默认关闭 IP 白名单，不限制调用 IP；开启后仅白名单内 IP 可调用，需设置外网 IP，保存后约 5 分钟生效。
接口鉴权
鉴权方法
在请求地址后拼接鉴权参数，示例 URL：wss://tts-api.xfyun.cn/v2/tts?authorization=xxx&date=xxx&host=tts-api.xfyun.cn
鉴权参数
表格
参数	类型	必须	说明
host	string	是	tts-api.xfyun.cn
date	string	是	RFC1123 格式 GMT 时间，允许 ±300 秒偏差
authorization	string	是	base64 编码的签名信息
authorization 生成步骤
获取 APIKey、APISecret（控制台 32 位字符串）。
拼接 signature_origin：
host: tts-api.xfyun.cn
date: Thu, 01 Aug 2019 01:53:21 GMT
GET /v2/tts HTTP/1.1
用 hmac-sha256+APISecret 签名，base64 编码得 signature。
拼接 authorization_origin：
api_key="xxx",algorithm="hmac-sha256",headers="host date request-line",signature="xxx"
base64 编码 authorization_origin 得到最终 authorization。
鉴权结果
成功：HTTP 101（协议升级）
失败：401/403，返回错误信息
接口数据传输与接收
握手成功后发送 JSON 格式数据，包含 common、business、data 三部分。
公共参数（common）
表格
参数	类型	必传	说明
app_id	string	是	平台申请的 APPID
业务参数（business）
表格
参数	类型	必传	说明
aue	string	是	音频编码：raw/pcm、lame/mp3、opus 等
sfl	int	否	mp3 需传 1，开启流式返回
auf	string	否	采样率：audio/L16;rate=16000/8000
vcn	string	是	发音人，需控制台开通
speed	int	否	语速 0-100，默认 50
volume	int	否	音量 0-100，默认 50
pitch	int	否	音高 0-100，默认 50
bgs	int	否	背景音 0 无 / 1 有，默认 0
tte	string	否	文本编码，默认 UTF8
数据参数（data）
表格
参数	类型	必传	说明
text	string	是	文本 base64 编码，＜8000 字节
status	int	是	固定 2
请求示例
{"common": {"app_id": "12345678"},"business": {"aue": "raw", "vcn": "xiaoyan", "speed": 50},"data": {"status": 2, "text": "base64 文本"}}
返回参数
表格
参数	说明
code	0 成功，其他异常
data.audio	base64 音频数据
data.status	1 合成中，2 合成结束
sid	会话 ID
注意事项
可忽略 data 为空且 code=0 的帧。
确保编码格式与 tte 参数一致。
超长文本分段调用。
发音人需在控制台开通权限。
常见错误码
10005：appid 授权失败
11200：无发音人权限
10161：base64 解码失败
401：签名错误
403：时钟偏移 / IP 白名单错误

## 🔄 数据库变化

### 新增表

#### 1. 家庭组表 (family_groups)

```sql
CREATE TABLE family_groups (
    id INT PRIMARY KEY AUTO_INCREMENT,
    group_name VARCHAR(100) NOT NULL COMMENT '家庭组名称',
    owner_id INT NOT NULL COMMENT '组长/创建者ID',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (owner_id) REFERENCES users(user_id)
);
```

#### 2. 家庭组申请表 (family_applications)

```sql
CREATE TABLE family_applications (
    id INT PRIMARY KEY AUTO_INCREMENT,
    family_id INT NOT NULL COMMENT '家庭组ID',
    user_id INT NOT NULL COMMENT '申请人ID',
    status ENUM('pending', 'approved', 'rejected') DEFAULT 'pending',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (family_id) REFERENCES family_groups(id),
    FOREIGN KEY (user_id) REFERENCES users(user_id)
);
```

### 修改表

#### users 表新增字段

```sql
ALTER TABLE users ADD COLUMN family_id INT NULL COMMENT '所属家庭组ID';
ALTER TABLE users ADD FOREIGN KEY (family_id) REFERENCES family_groups(id);
```

## 📡 新增接口详解

### 一、家庭组管理接口

#### 1. 创建家庭组

**接口地址：** `POST /api/family/create`

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| name | string | 是 | 家庭组名称（Query参数） |

**请求示例：**

```javascript
// 方式1：Query参数
POST /api/family/create?name=我的家庭组

// 方式2：JavaScript
const response = await fetch('/api/family/create?name=我的家庭组', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`
  }
});
```

**响应示例：**

```json
{
  "code": 200,
  "message": "家庭组创建成功",
  "data": {
    "family_id": 1,
    "group_name": "我的家庭组"
  }
}
```

**说明：**
- 创建者自动成为管理员
- 创建者自动加入该家庭组（user.family_id 自动绑定）
- 用户只能在一个家庭组中

---

#### 2. 申请加入家庭组

**接口地址：** `POST /api/family/{family_id}/apply`

**路径参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| family_id | int | 是 | 家庭组ID |

**请求示例：**

```javascript
// 用户输入家庭组ID后调用
const familyId = 1;  // 用户输入的ID

const response = await fetch(`/api/family/${familyId}/apply`, {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`
  }
});
```

**响应示例：**

```json
{
  "code": 200,
  "message": "申请已发送，等待管理员审批",
  "data": null
}
```

**错误响应：**

```json
// 家庭组不存在
{
  "code": 404,
  "message": "未找到该ID的家庭组",
  "data": null
}

// 重复申请
{
  "code": 400,
  "message": "您已提交过申请，请等待管理员审批",
  "data": null
}
```

---

#### 3. 查看待审批列表（管理员）

**接口地址：** `GET /api/family/applications`

**请求示例：**

```javascript
const response = await fetch('/api/family/applications', {
  method: 'GET',
  headers: {
    'Authorization': `Bearer ${token}`
  }
});
```

**响应示例：**

```json
{
  "code": 200,
  "message": "获取成功",
  "data": [
    {
      "application_id": 1,
      "user_id": 2,
      "phone": "13800138000",
      "apply_time": "2026-02-23 10:30:00"
    },
    {
      "application_id": 2,
      "user_id": 3,
      "phone": "13900139000",
      "apply_time": "2026-02-23 11:00:00"
    }
  ]
}
```

**说明：**
- 只有家庭组管理员可以调用
- 返回所有待审批（pending）的申请
- 包含申请人的基本信息

---

#### 4. 审批申请（管理员）

**接口地址：** `PUT /api/family/applications/{app_id}`

**路径参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| app_id | int | 是 | 申请记录ID |

**Query参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| is_approve | bool | 是 | true=同意，false=拒绝 |

**请求示例：**

```javascript
// 同意申请
const response = await fetch('/api/family/applications/1?is_approve=true', {
  method: 'PUT',
  headers: {
    'Authorization': `Bearer ${token}`
  }
});

// 拒绝申请
const response = await fetch('/api/family/applications/1?is_approve=false', {
  method: 'PUT',
  headers: {
    'Authorization': `Bearer ${token}`
  }
});
```

**响应示例：**

```json
{
  "code": 200,
  "message": "操作成功，已同意",
  "data": null
}
```

**说明：**
- 只有家庭组管理员可以审批
- 同意后，申请人的 family_id 自动绑定
- 申请状态更新为 approved 或 rejected

## 💻 前端使用示例

### 1. 家庭组管理流程

```javascript
class FamilyManager {
  constructor(token) {
    this.token = token;
    this.baseUrl = 'http://localhost:8000';
  }
  
  // 创建家庭组
  async createFamily(name) {
    const response = await fetch(
      `${this.baseUrl}/api/family/create?name=${encodeURIComponent(name)}`,
      {
        method: 'POST',
        headers: {
          'Authorization': `Bearer ${this.token}`
        }
      }
    );
    return await response.json();
  }
  
  // 申请加入家庭组
  async applyToJoin(familyId) {
    const response = await fetch(
      `${this.baseUrl}/api/family/${familyId}/apply`,
      {
        method: 'POST',
        headers: {
          'Authorization': `Bearer ${this.token}`
        }
      }
    );
    return await response.json();
  }
  
  // 获取待审批列表（管理员）
  async getApplications() {
    const response = await fetch(
      `${this.baseUrl}/api/family/applications`,
      {
        method: 'GET',
        headers: {
          'Authorization': `Bearer ${this.token}`
        }
      }
    );
    return await response.json();
  }
  
  // 审批申请（管理员）
  async reviewApplication(appId, isApprove) {
    const response = await fetch(
      `${this.baseUrl}/api/family/applications/${appId}?is_approve=${isApprove}`,
      {
        method: 'PUT',
        headers: {
          'Authorization': `Bearer ${this.token}`
        }
      }
    );
    return await response.json();
  }
}

// 使用示例
const familyManager = new FamilyManager('your_jwt_token');

// 创建家庭组
const result = await familyManager.createFamily('我的家庭组');
console.log('家庭组ID:', result.data.family_id);

// 申请加入
await familyManager.applyToJoin(1);

// 查看申请（管理员）
const applications = await familyManager.getApplications();
console.log('待审批:', applications.data);

// 同意申请（管理员）
await familyManager.reviewApplication(1, true);
```

### 2. 安全报告生成

```javascript
class SecurityReportManager {
  constructor(token) {
    this.token = token;
    this.baseUrl = 'http://localhost:8000';
  }
  
  // 获取安全报告
  async getSecurityReport(userId) {
    const response = await fetch(
      `${this.baseUrl}/api/users/${userId}/security-report`,
      {
        method: 'GET',
        headers: {
          'Authorization': `Bearer ${this.token}`
        }
      }
    );
    return await response.json();
  }
  
  // 渲染报告到页面
  async renderReport(userId, containerId) {
    const data = await this.getSecurityReport(userId);
    
    // 使用Markdown渲染库
    const md = new MarkdownIt();
    const htmlContent = md.render(data.report_content);
    
    // 渲染到页面
    document.getElementById(containerId).innerHTML = htmlContent;
    
    return data;
  }
}

// 使用示例
const reportManager = new SecurityReportManager('your_jwt_token');

// 获取并渲染报告
await reportManager.renderReport(1, 'report-container');
```

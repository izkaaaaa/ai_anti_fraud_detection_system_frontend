## 🐛 常见问题排查

### 问题 1：通知不显示

**原因**：通知权限未授予

**解决**：
1. 进入系统设置 → 应用 → 你们的 App → 通知
2. 开启通知权限
3. 重新启动监测

---

### 问题 2：后台录音仍然失效

**原因**：前台服务被系统杀死

**解决**：
1. 检查日志中是否有"前台服务已启动"
2. 检查通知栏是否有通知
3. 进入系统设置 → 电池 → 应用耗电管理
4. 将你们的 App 设置为"不限制"

---

### 问题 3：编译错误

**错误信息**：
```
Error: The method 'FlutterForegroundTask.init' isn't defined
```

**解决**：
```bash
flutter clean
flutter pub get
flutter run
```

---

### 问题 4：前台服务启动失败

**日志显示**：
```
❌ 前台服务启动失败
```

**解决**：
1. 检查 AndroidManifest.xml 中是否有服务声明
2. 检查是否授予了通知权限
3. 重启 App

## 📱 不同手机厂商的注意事项

### 小米 MIUI
- 需要在"自启动管理"中允许你们的 App 自启动
- 需要在"省电优化"中关闭对你们 App 的优化

### 华为 EMUI
- 需要在"应用启动管理"中设置为"手动管理"
- 允许"后台活动"

### OPPO ColorOS
- 需要在"应用管理"中允许"后台运行"
- 关闭"智能省电"

### vivo FuntouchOS
- 需要在"后台高耗电"中允许你们的 App
- 关闭"后台冻结"

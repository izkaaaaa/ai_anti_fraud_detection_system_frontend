# WebSocket 检测结果接口问题反馈

**日期：** 2026-03-13  
**反馈方：** 前端团队  
**问题状态：** 待后端修复

---

## 一、问题概述

前端实时监测页面中，音频/视频/文本三个检测模块的**置信度百分比始终显示为 0%**。  
经前端排查，前端接口解析逻辑正确，问题根源在于后端 WebSocket 推送的消息中 `confidence` 字段值始终为 `0.0`。

---

## 二、实际收到的消息（前端日志）

### 消息 1：`detection_result` 类型

```json
{
  "type": "detection_result",
  "detection_type": "语音",
  "is_risk": false,
  "confidence": 0.0,
  "message": "安全",
  "timestamp": ""
}
```

**问题：**
- `confidence` 字段值为 `0.0`，不是实际检测置信度
- `timestamp` 字段为空字符串，没有时间信息
- `message` 字段内容过于简短（仅"安全"），没有实质信息

### 消息 2：`info` 类型（同一事件再次推送）

```json
{
  "type": "info",
  "data": {
    "title": "语音检测通过",
    "message": "当前通话环境安全 (置信度: 0.00)。",
    "risk_level": "safe",
    "confidence": 0.0,
    "timestamp": "2026-03-13T11:09:34.349513"
  }
}
```

**问题：**
- `data.confidence` 字段同样为 `0.0`
- 真实置信度被拼接在了 `data.message` 文字里（`"置信度: 0.00"`），而非作为独立数值字段
- **同一个检测事件发送了两条消息**（`detection_result` + `info`），造成前端重复处理

---

## 三、前端期望的消息格式

### 方案 A：统一使用 `detection_result` 格式（推荐）

```json
{
  "type": "detection_result",
  "detection_type": "语音",
  "is_risk": false,
  "confidence": 0.85,
  "message": "当前通话环境安全，未检测到异常语音特征",
  "timestamp": "2026-03-13T11:09:34.349513"
}
```

### 方案 B：统一使用 `info` 格式

```json
{
  "type": "info",
  "data": {
    "title": "语音检测通过",
    "message": "当前通话环境安全，未检测到异常语音特征",
    "risk_level": "safe",
    "confidence": 0.85,
    "timestamp": "2026-03-13T11:09:34.349513"
  }
}
```

**无论选哪种方案，请确保：**
1. `confidence` 是实际的浮点数值（范围 `0.0 ~ 1.0`），不要固定为 `0.0`
2. `timestamp` 字段有正确的时间戳，不要为空字符串
3. 同一检测事件只推送一条消息，不要同时发 `detection_result` 和 `info` 两条

---

## 四、前端当前接口解析逻辑（供参考）

前端目前同时兼容两种消息格式，解析规则如下：

### `detection_result` 解析

| 后端字段 | 前端使用方式 | 说明 |
|---------|------------|------|
| `detection_type` | 区分语音/视频/文本 | 支持中文（`语音`/`视频`/`文本`）和英文（`audio`/`video`/`text`） |
| `is_risk` | 判断是否有风险 | `bool` 类型 |
| `confidence` | 显示为百分比进度条 | `float`，范围 `0.0~1.0` |
| `message` | 展示检测说明文字 | `string` |
| `timestamp` | 显示检测时间 | ISO 8601 格式字符串 |

### `info` 解析

| 后端字段 | 前端使用方式 | 说明 |
|---------|------------|------|
| `data.title` | 判断检测类型（含"语音"→语音，含"视频"→视频，否则→文本） | `string` |
| `data.risk_level` | 判断是否有风险（非`safe`即风险） | `string`：`safe` / `medium` / `high` |
| `data.confidence` | 显示为百分比进度条 | `float`，范围 `0.0~1.0` |
| `data.message` | 展示检测说明文字 | `string` |
| `data.timestamp` | 显示检测时间 | ISO 8601 格式字符串 |
